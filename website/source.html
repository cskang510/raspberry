Source Code

All of the source code is managed on GitHub, and freely available under the GPL license.

https://github.com/terrenjpeterson/raspberry

Charts

The charts are from a Google Framework that renders both line graphs as well as gauges that represent different key data about growing the plants.

AWS Integration

The sensor scripts leverage a SDK from AWS called Boto that enables invoking different API's directly from the Raspberry Pi. This is python based and examples of different functions include uploading photos directly from a RaPi.

Twitter Integration

The Twitter Apps framework is used to post tweets to the social media site directly from the Raspberry Pi using different API's.
